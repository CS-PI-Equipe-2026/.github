# ⚓ Bem-vindo à organização Argonautas!

Somos a equipe **Argonautas**, formada por alunos do [Campus Paranavaí do Instituto Federal do Paraná (IFPR)](https://ifpr.edu.br/paranavai/).

Nossa organização foi criada para o desenvolvimento de projetos nas disciplinas de **Projeto Integrador** e **Construção de Software**, ministradas pelos professores Frank e Marcelo. Nosso objetivo é entregar soluções tecnológicas eficientes, aplicando metodologias ágeis e boas práticas de engenharia de software.
